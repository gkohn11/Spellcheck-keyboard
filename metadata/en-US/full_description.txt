This keyboard is created for those who only need a keyboard and nothing more.

To enable the keyboard:

* Open "Simple Spellcheck" from your launcher
* Enable Simple Spellcheck (default system warning about tracking will be shown)
* Switch to Simple Spellcheck from current Input Method (differs between keyboards, usually long-press space)
* To edit Simple Spellcheck settings long-press "," or open system Settings, Languages &amp; Input, Simple Spellcheck.
* You can enable/disable all the input methods in Settings, Languages &amp; Input, Manage Keyboards (differs between phones)

<b>Features:</b>

* Small size (&lt;1MB)
* Adjustable keyboard height for more screen space
* Number row
* Swipe space to move pointer
* Delete swipe
* Custom theme colors
* Minimal permissions (only Vibrate)
* Ads-free
